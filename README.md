# Creative Video Editor Portfolio Website

A clean, modern, light-themed portfolio website for Radha Kushwaha - a Creative Video Editor specializing in social media content, meme editing, and YouTube video production.

## 🎯 Project Overview

This is a professional, minimal, single-page portfolio website designed to showcase video editing work with a fast-loading, mobile-responsive design. The website features a soft off-white/beige color scheme for a premium and professional look.
